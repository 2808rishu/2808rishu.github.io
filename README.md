# Ridham Jain's Personal Website

This is the personal website/portfolio of Ridham Jain, showcasing education, skills, experience, and certifications.

## Features

- Responsive design that works on all devices
- Dark theme with cool red accents
- Interactive tab navigation
- Embedded PDF viewer for certificates
- Downloadable resume

## Sections

- **Home**: Introduction and summary of skills
- **Education**: Educational background, certifications, and positions of responsibility
- **Skills**: Technical and soft skills
- **Experience**: Professional experience
- **Hobbies**: Personal interests
- **Contact**: Contact information and resume download

## Technologies Used

- HTML5
- CSS3
- JavaScript

## Deployment

This website is hosted on GitHub Pages and automatically deployed from the main branch.
